# Logos

This directory contains logo files (SVG/PNG) extracted from the Figma design system.

## Figma Source

- **Node ID:** `33-587`
- **File:** `ixkN3RSoiQc5aSlxZ21Hay`

## Usage

Logos should be exported from Figma and placed in this directory.

### Exporting from Figma

1. Select the logo component in Figma
2. Right-click and select "Export"
3. Choose SVG or PNG format
4. Save to this directory with a descriptive name

### Logo Variants

Include all logo variants:
- `logo-full.svg` - Full logo with text
- `logo-mark.svg` - Logo mark only
- `logo-light.svg` - Light version
- `logo-dark.svg` - Dark version

### Using Logos

```html
<!-- Full logo -->
<img src="./assets/logos/logo-full.svg" alt="Vallejo Logo">

<!-- Logo mark -->
<img src="./assets/logos/logo-mark.svg" alt="Vallejo">
```
