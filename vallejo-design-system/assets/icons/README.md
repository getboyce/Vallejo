# Icons

This directory contains SVG icon files extracted from the Figma design system.

## Usage

Icons should be exported from Figma as SVG files and placed in this directory.

### Exporting from Figma

1. Select the icon component in Figma
2. Right-click and select "Export"
3. Choose SVG format
4. Save to this directory with a descriptive name (e.g., `icon-check.svg`, `icon-arrow-right.svg`)

### Using Icons

```html
<!-- Inline SVG -->
<img src="./assets/icons/icon-check.svg" alt="Check icon">

<!-- Or use as background -->
<div style="background-image: url('./assets/icons/icon-check.svg');"></div>
```

## Naming Convention

- Use lowercase
- Separate words with hyphens
- Prefix with `icon-` (e.g., `icon-check.svg`, `icon-arrow-right.svg`)
